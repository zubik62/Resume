a. cd
   Переход в домашнюю директорию

b. mkdir nordic
   Создание новой папки

c. cd nordic/
   Переходв папку

d. touch nordic.txt
   Создание нового файла

e. vim nordic.txt ; клавиша i 
   Переход в текстовый редактор ; переход в режим редактирования, набор текста
   
f. клавиша Esc ; :wq
   Переключение в командный режим ; запись измененний и выход
   
g. cat text.txt | grep nordic
   Просмотр содержимого файла и поиск по слову nordic 

h. cat text.txt | grep nordic >> text2.txt
   Просмотр содержимого файла, поиск по слову nordic и перенаправленияв новый файл
i. ls
   Просмотр файлов в текущем каталоге
   
j. ls -a
   Просмотр файлов, включая скрытые в текущем каталоге

k. pwd
   Показывает путь текущего каталога

l.  mkdir test3 test4 && cd test3/ && touch text2.txt && cp ./text2.txt ../test4
   Создание двух папок, переход в папку, создание там файла, копирование файла в другую папку
